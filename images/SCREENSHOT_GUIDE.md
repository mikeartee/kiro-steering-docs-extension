# Screenshot Guide

Please capture the following screenshots for the README documentation:

## Required Screenshots

### 1. Extension in Kiro Sidebar (`sidebar-view.png`)

- Open Kiro IDE
- Show the Steering Documents view in the Kiro sidebar
- Capture the full sidebar with document categories expanded
- Show both installed and available documents

### 2. Toggle Functionality (`toggle-demo.png`)
- Show a document with the toggle button visible
- Capture before/after states or use annotation to show the toggle action
- Highlight the toggle icon (circle-outline/check)

### 3. Document Preview (`preview-demo.png`)
- Right-click on a document and select "Preview Document"
- Capture the preview pane showing document content
- Show the markdown rendering

### 4. Installation Confirmation (`install-notification.png`)
- Capture the notification that appears after toggling a document on
- Show "Document activated" or similar success message

## Screenshot Specifications

- Format: PNG
- Resolution: High DPI (2x or higher for clarity)
- Size: Keep under 500KB each if possible
- Crop: Remove unnecessary UI elements, focus on the extension

## Taking Screenshots in VS Code/Kiro

1. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
2. Type "Screenshot" or use your OS screenshot tool
3. Crop to relevant area
4. Save to this `images/` folder with the names above

## After Taking Screenshots

1. Delete this guide file
2. Update the README.md to reference the screenshots
3. Verify all images display correctly in the README preview
